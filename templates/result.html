Feature Abbreviations Explained

  • Num_Loops: Number of Loops - Total count of loops (for, while) in the code.
  • Num_Variables: Number of Variables - Total variables declared in the code.
  • Num_Comments: Number of Comments - Total comment lines in the code.
  • Code_Complexity: Cyclomatic Complexity - Measure of code's control flow complexity.
  • Lines_Code: Lines of Code - Total lines present in the source file.
  • Ratio_Comment_Code: Ratio of Comments to Code - Proportion of comments relative to code lines.
  • Max_Nesting: Maximum Nesting Depth - Deepest level of nested blocks or loops.
  • Num_Functions: Number of Functions - Count of function/method definitions.
  • Avg_Function_Length: Average Function Length - Mean number of lines per function.
  • Num_Classes: Number of Classes - Count of class definitions.
  • Max_Function_Params: Max Function Parameters - Highest number of parameters in any function.
  • Fan_In: Fan-In - Number of functions calling a given function.
  • Fan_Out: Fan-Out - Number of functions called by a given function.
  • Num_Statements: Number of Statements - Total executable statements in the code.
  • Num_Imports: Number of Imports - Count of imported modules or libraries.
  • Halstead_Volume: Halstead Volume - Complexity metric based on operators and operands.
